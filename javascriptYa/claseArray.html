Clase Array

Descripción

Un arreglo o vector es una estructura de datos que permite almacenar un conjunto de datos. Con un único nombre se define un vector y por medio de un subíndice hacemos referencua a cada elemento del mismo. Para crear un array de n elementos, la sintaxis es la siguiente:

var mi_arreglo;
mi_arreglo = new Array(n);
Donde mi_arreglo es nuestro arreglo y n es la dimensión del arreglo.

Para acceder a los diferentes índices de un arreglo, ya sea para modificar su valor o para obtenerlo, basta con hacer referencia al índice que necesitamos.

Ejemplo

Crearemos un programa que pida al usuario los nombres y sueldos de n empleados (cantidad a elegir por el usuario), mismos que se guardarán en dos arreglos distintos a través de una función set. Posteriormente se deberá mostrar en una tabla el sueldo de cada empleado, así como el total de la suma de todos los sueldos. Adicionalmente se agrega estilo a la tabla y se hace a prueba de errores (en caso de no meter números en la petición) solo por diversión. Ejecutar ejemplo
