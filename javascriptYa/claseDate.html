Clase Date

Descrićión

Javascript dispone de varias clases predefinidas para acceder a muchas de las funciones normales de cualquier lenguaje, como puede ser el manejo de vectores o el de fechas. La clase code nos permitirá manejar fechas y horas. Puede ser invocada de la siguiente forma:

fecha = new Date(); //creación de un objeto de la clase Date
fecha = new Date(año, mes, dia);
fecha = new Date(año, mes, dia, hora, minuto, segundo);
Si no utilizamos parámetros, el objeto fecha contendrá la fecha y hora actual recuperadas del reloj de nuestra computadora. En caso contrario, hay que tener en cuenta que los meses comienzan por cero. Así, por ejemplo:
navidad2020 = new Date(2020, 11, 25);
El objeto Date dispone, entre otros, de los siguientes métodos:
  • getFullYear()
  • setFullYear(año)
  • getMonth
  • setMonth(mes)
  • getDate() //Día del mes
  • setDate(dia)
  • getHours()
  • setHours(horas)
  • getMinutes()
  • setMinutes(minutos)
  • getSeconds()
  • setSeconds(segundos)
  • getDay() //Día de la semana (0 - 6)
Llegado este punto es importante aclarar que en muchas de las clases que creemos será útil crear métodos set y get. Los métodos get sirven para obtener el valor de los atributos de un objeto, mientras que los métodos set sirven para definirlos (darles un valor).
