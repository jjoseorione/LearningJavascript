Entrada de datos por teclado

Descripción

Para la entrada de datos por teclado existe la función prompt. Cada vez que necesitamos ingresar un dato con ésta función, aparece una ventana donde cargamos el valor. Hay otras formas más sofisticadas para la entrada de datos en un documento HTML, pero el apendizaje de los conceptos básicos de Javascript nos resultará más práctica esta función.

Para usar la función prompt, debemos igual la variable que queremos que guarde la entrada por teclado a la función. La función prompt requiere de dos argumentos separados por coma; el primer argumento es el mensaje a mostrar, mientras que el segundo es el valor inicial que mostrará la variable.
