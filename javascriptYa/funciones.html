Funciones

Descripción

Las funciones son porciones de código que podemos creas fuera del flujo normal del programa para luego invocarlas cuando sea necesario. Esto es de gran utilidad en todos los lenguajes de programación, ya que de esta manera podemos escribir sólo una vez una porción de código que va a repetirse en varios lugares y en varias ocasiones en el flujo normal del programa y posteriormente sólo invocar esa porción de código, en lugar de escribirla más de una vez.

Al igual que en otros lenguajes de programación, las funciones pueden hacer uso de parámetros y devolver valores.

Para declarar una función la sintaxis es la siguiente:

function nombreFunción(par1, par2, ... , parN)
{
  instrucciones;
  return var;
}
Donde nombreFunción es el nombre que nosotros daremos a nuestra función; par1, par2, parN son los parámetros que mandaremos a nuestra función para que pueda operar con ellos, los cuales deben ir separado por comas; instrucciones son las líneas de código que formarán parte de nuestra función y var es la variable a ser retornada. Las palabras reservadas function es obligatoria, al igual que return (en caso de que queramos retornar un valor).
