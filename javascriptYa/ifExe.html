Ejemplo de condiciones if

Este programa lee 3 números enteros. Si el primer número ingresado es el mayor de los tres, se muestra la suma de los tres; si el segundo es el más grande, se muestra el producto; si el tercero es el mayor, se muestran la suma y el producto; si existe un empate entre dos o más por ser el mayor, se tomará en cuenta el primero en ser ingresado. Además, el programa también nos indica si existe uno mayor o si existe un empate.
