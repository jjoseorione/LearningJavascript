Selector switch

Descripción

Un selector switch es similar a un condiconal if, con la diferencia de que el switch está diseñado previamente para poder elegir entre más de dos casos, a diferencia del if que sólo permite elegir entre dos casos distintos (a menos que anidemos varios ifs).

La sintaxis del selector switch es la siguiente:

Switch(var)
{
  case valor1: intrucciones
               break;

  case valor2: intrucciones
               break;

  ...

  default: intrucciones
}
Donde var es la variable a evaluar y valor1, valor2, etc., son los valores que puede tomar. La instrucción break sirve para salir del ciclo, lo que significa que ya se evaluó la opción deseada. El segmento default se ejecutará cuando no se coincida con ninguno de los casos. Se puede omitir el segmento default y, de esta forma, si no se cumple ninguno de los casos no se realizará ninguna instrucción.

La desventaja de un switch con respecto a un if es que switch sólo evalúa una variable a la vez.
