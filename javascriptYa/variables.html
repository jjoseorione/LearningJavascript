Variables

Descripción

Como sabemos, una variable es símplemente una locación de memoria donde se guarda un valor. Los principales tipos de variables que pueden ser guardadas en Javascript son los siguientes:

  • Enteros. Por ejempplo: 100, 260, 43, etc.
  • Reales. Por ejemplo: 1.24, 69.90, 3.141592, etc.
  • Cadenas de caracteres. Por ejemplo: 'Juan', 'Compras', 'Listado', etc.
  • Booleanos. Es decir, 'true' y 'false'.

Para declarar una variable únicamente debe ponerse la palabra var, seguida del nombre de nuestra variable y separadas por un espacio. Al igual que en otros lenguajes, como C, es posible declarar más de una variable en una misma linea agregando más nombrs de variables separadas por comas. También, como ocurre en otros lenguajes, es posible inicializar una variable al momento de su declaración, o bien, puede inicializarse después.

Para imprimir una variable en nuestro documento necesitamos nuevamente la función write del objeto document. Para imprimir el valor de una variable, omitimos las comillas simples, ya que si las usamos aparecería el nombre en lugar del valor.

Cuando imprimimos varios datos es posible concatenar con el operador '+' para que no sea necesario llamar a la función write más de una vez.

Los comentarios, al igual que en C pueden ponerse por línea con una doble diagonal (//) o por bloque con diagonal y asterísco (/* */).
